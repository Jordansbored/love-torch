# Love Torch

A Valentine's Day themed Hytale mod that adds a romantic torch with heart particles.

## Overview

The **Love Torch** features a lovely pink flame accompanied by a spiraling tornado of red and pink hearts. Perfect for Valentine's Day builds or adding a touch of romance to your world.

| Torch | Effect | Light Color |
|-------|--------|-------------|
| **Love Torch** | Pink flame with spiral heart tornado | Pink (#ff69b4) |

## Installation

1. Download the latest `LoveTorch-1.0.0.jar` from [Releases](https://github.com/Jordansbored/love-torch/releases).
2. Place the JAR in your Hytale mods folder:
   - **Linux (Flatpak):** `~/.var/app/com.hypixel.HytaleLauncher/data/Hytale/UserData/Mods/`
   - **Windows:** `%AppData%/Hytale/UserData/Mods/`
3. Restart Hytale and enable the mod.

## Crafting Recipe

Crafted at a **Workbench** using basic resources:

**Input:**
- 4x **Crude Torch** (`Furniture_Crude_Torch`)

**Output:**
- 4x **Love Torch**

## Building from Source

To build the mod from source:

```bash
./gradlew build
```

The built JAR will be located at `build/libs/LoveTorch-1.0.0.jar`.

## Features

- Custom pink fire particle system.
- Spiraling heart tornado effect.
- Wall placement support.
- Toggle on/off with right-click interaction.
- Fully integrated with Hytale's lighting system.
